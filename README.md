This repository is an introductory group project designed to help students become familiar with GitHub and collaborative workflows.  
It is used to practice creating repositories, working with branches, committing changes, and submitting pull requests.

## Assignment Overview
As part of this assignment, the group collaboratively:
1. Created a shared GitHub repository
2. Added a common file (`hello_ml.txt`)
3. Used individual branches for each member
4. Submitted changes via Pull Requests
5. Reviewed and merged all contributions into the main branch

The final file contains a greeting from each group member as proof of collaboration.
